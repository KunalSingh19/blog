Onex is a package manager for hacker's. Onex manage more than 400+ hacking tools that can be installed on single click.

Requirements :

 - Termux Application from F-Droid
 - Python Package

Commands :

 $ pkg install python -y
 $ pip install onex
 $ onex

Options :

 $ onex [-h|i|v|a|r]

options :-
-h : print help
-i : install
-v : version
-a : about us
-r : remove
